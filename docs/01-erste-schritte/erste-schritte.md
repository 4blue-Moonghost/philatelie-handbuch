# Erster Start

Nach der Installation startet der **Einrichtungsassistent** automatisch.

## Einrichtungsassistent
![Übersicht Dashboard](/img/screenshots/dashboard.png)


Beim ersten Start kannst du folgende Optionen festlegen:

- **Automatische Updates** — Empfohlen aktiviert lassen
- **Telemetrie** — Anonyme Nutzungsstatistiken (optional)
- **Autostart** — App beim Windows-Start automatisch öffnen

## Sprache und Einstellungen

Die App erkennt automatisch deine Systemsprache. Du kannst die Sprache jederzeit unter **Einstellungen → Allgemein** ändern.

## Lizenz aktivieren

Die App startet im **Free-Modus** (max. 100 Artikel, 2 Bilder pro Artikel).  
Für unbegrenzte Nutzung aktiviere eine Lizenz unter **Lizenz aktivieren**.
