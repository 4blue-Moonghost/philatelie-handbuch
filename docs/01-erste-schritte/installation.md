# Installation

## Windows

1. Lade die aktuelle Version von [philatelie-manager.com](https://philatelie-manager.com) herunter
2. Starte die `Philatelie-Manager-Setup-x.x.x.exe`
3. Folge dem Installationsassistenten
4. Die App startet automatisch nach der Installation

:::info Windows SmartScreen
Falls Windows SmartScreen eine Warnung anzeigt, klicke auf **"Weitere Informationen"** → **"Trotzdem ausführen"**. Die App ist sicher, das EV Code-Signing Zertifikat wird demnächst eingebunden.
:::

## macOS

1. Lade die `.dmg` Datei herunter
2. Öffne die DMG-Datei und ziehe die App in den Programme-Ordner
3. Beim ersten Start: Rechtsklick → Öffnen (einmalig nötig)
