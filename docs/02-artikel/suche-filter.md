# Suche & Filter

## Volltextsuche

Die Suchleiste findet Artikel nach:
- Titel
- Beschreibung
- Michel-Nummer
- Motiv/Thema
- Notizen
- **eBay-Artikelnummer**

## Filter

| Filter | Optionen |
|---|---|
| **Kategorie** | Briefmarke, Block, Bogen, FDC, Postkarte, Brief, Lot, Sonstiges |
| **Status** | Entwurf, Bei eBay online, Verkauft, Versendet |
| **Land** | Alle verfügbaren Länder |
