# eBay einrichten

## eBay-Account verbinden

1. Gehe zu **Einstellungen → eBay**
2. Klicke auf **Mit eBay verbinden**
3. Es öffnet sich das eBay-Login-Fenster
4. Melde dich an und erteile die Berechtigung
5. Der Token wird automatisch gespeichert

:::info Token-Gültigkeit
Der eBay-Token ist 18 Monate gültig. Nach Ablauf musst du dich erneut verbinden.
:::

## Einstellungen

| Einstellung | Beschreibung |
|---|---|
| **Verkäufer-Typ** | Privat oder Gewerblich |
| **Versandkosten** | Standard-Versandkosten für neue Listings |
| **Rückgaberichtlinie** | Aus deinem eBay-Konto |
