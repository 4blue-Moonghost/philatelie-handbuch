# Versandlabels erstellen

:::info Demnächst verfügbar
Die direkte Label-Erstellung ist in Entwicklung. Aktuell werden Versandkosten berechnet und du wirst zur jeweiligen Anbieter-Seite weitergeleitet.
:::

## Geplante Features

- **DHL Paketlabels** direkt in der App erstellen
- **Deutsche Post Internetmarke** für Briefe und Warensendungen
- **Sendcloud** Multi-Carrier Label-Druck
- **Label-History** — alle erstellten Labels im Überblick

## Aktueller Workaround

1. Versandkosten in der App berechnen
2. Label direkt beim Anbieter erstellen:
   - [DHL Geschäftskundenportal](https://geschaeftskunden.dhl.de)
   - [Deutsche Post Frankierung](https://www.deutschepost.de)
   - [Sendcloud](https://www.sendcloud.de)
