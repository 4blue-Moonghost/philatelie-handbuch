# Briefmarken scannen

## Einzelscan

1. Lege die Briefmarke auf den Scanner
2. Wähle den Scanner aus dem Dropdown
3. Wähle das **DPI-Profil** (300 DPI für Briefmarken empfohlen)
4. Klicke **Scannen**

## ADF-Scan (Mehrseitenscan)

Für Briefe oder FDCs mit mehreren Seiten:
1. Lege die Dokumente in den ADF
2. Aktiviere **Mehrseiten-Scan**
3. Alle Seiten werden als Grid-Vorschau angezeigt
4. Wähle welche Seiten du übernehmen möchtest

## Scan einem Artikel zuweisen

Nach dem Scan kannst du das Bild direkt einem neuen oder bestehenden Artikel zuweisen.
